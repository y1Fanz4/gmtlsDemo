# gmtls-demo 使用说明

## 环境要求

- Go 1.21+

## 快速开始

### 第一步：生成国密证书

```bash
cd certgen
go run main.go
```

生成后 `certs/` 目录结构：

```
certs/
├── ca.crt / ca.key            # CA 根证书
├── server_sign.crt/key        # 服务端签名证书（用于握手认证）
├── server_enc.crt/key         # 服务端加密证书（用于密钥交换）
└── client.crt / client.key    # 客户端证书
```

### 第二步：启动服务端

```bash
cd server
go run main.go
```

输出示例：

```
========================================
  国密 TLS 服务端已启动
  监听地址: localhost:8443
========================================
```

### 第三步：启动客户端

新开一个终端：

```bash
cd client
go run main.go
```

输出示例：

```
========================================
  正在连接国密 TLS 服务端...
  地址: localhost:8443
  协议: GMSSL 1.0 (SM2+SM3+SM4)
========================================

✓ 国密 TLS 握手成功
  密码套件: 0xE013
  服务端证书 CN: localhost

  输入消息后回车发送，输入 'exit' 退出
========================================

请输入要发送的消息 > hello
----------------------------------------
  服务端响应 (20 字节):
  服务端已收到消息[hello]
----------------------------------------

请输入要发送的消息 > exit
```

## 注意事项

- 每次重新生成证书后需重启服务端
- 本项目仅供学习演示，生产环境请使用经过安全审计的实现
